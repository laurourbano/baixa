# Integração com S3 (Backups Persistentes)

Este documento descreve como configurar o backend para enviar backups para um bucket S3.

## Variáveis de ambiente necessárias

- `AWS_ACCESS_KEY_ID` — chave de acesso AWS
- `AWS_SECRET_ACCESS_KEY` — segredo da AWS
- `AWS_REGION` ou `AWS_DEFAULT_REGION` — região do bucket
- `S3_BUCKET_NAME` (ou `S3_BUCKET`) — nome do bucket onde os backups serão salvos

## Comportamento

Quando as variáveis acima estiverem definidas, o backend (`bridge/server.js`) tentará enviar cada backup para o caminho `backups/<backup-name>.json` no bucket configurado, além de gravar localmente.

## Exemplo

Defina as variáveis de ambiente no painel do Netlify (Site settings > Environment variables) ou no provedor escolhido. Não armazene credenciais diretamente no repositório.
